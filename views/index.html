API Doc

for REST Service Prüfungen

Get
Alle Prüfungen

Um deine bereits angelegten Prüfungen abrufen zu können, musst du nichts weiter als /api/pruefungen aufrufen.


Eine Prüfung

Möchtest du eine einzelne Prüfung betrachten, sende ihre id am Ende folgender Adresse: /api/pruefungen/

Post

Wenn du eine neue Prüfung anlegen möchtest, musst eine POST Request an /api/pruefungen schicken. Im Body deiner Anfrage übergibst du ein JSON-Objekt. Dieses muss folgende Struktur aufweisen:

{
"fach":"Verteilte Systeme",
"art":"Portfolio",
"datum":"4.3.2020",
"ort":"Raum 3.2.1"
}

Dabei ist jedes der Felder ein Pflichtfeld, abgesehen von "ort".

Put

Falls eine Prüfung geupdated werden muss, schicke eine PUT Request an /api/pruefungen/ . Damit die richtige Prüfung gefunden werden kann, musst du die entsprechende id der Prüfung am Ende der Adresse übergeben. Anschließend sendest du diese mitsamt deiner aktualisierten Daten (JSON-Objekt, wie im Code-Beispiel unter "Post") an die Schnittstelle. Beachte das du für alle Parameter bis auf "fach" neue Werte übergeben kannst.

Delete

Um eine Prüfung aus der Datenbank zu entfernen, wird ein DELETE Request verlangt. Übergebe dazu die entsprechende id der Prüfung am Ende folgender Adresse: /api/pruefungen/
